Catalog management
==================

The ACCESS-NRI Intake catalog is an 
`intake-dataframe-catalog <https://intake-dataframe-catalog.readthedocs.io/en/latest/?badge=latest>`_ that 
is managed using a purpose-built Python package called access-nri-intake. Instructions for how to install 
access-nri-intake can be found in the :ref:`installation` section. Here we describe the different components 
of access-nri-intake and how they can be used to build and manage the catalog.
