Using the catalog
=================

The ACCESS-NRI Intake catalog provides a way for Python users to peruse, search and load ACCESS and 
ACCESS-related climate data available on the Australian NCI supercomputer, Gadi. In the following 
sections we give an overview of the catalog and demonstrate how to use it.
